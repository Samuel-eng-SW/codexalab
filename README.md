# Codexalab

Blog técnico pessoal em desenvolvimento, voltado para estudos e reflexões
sobre Tecnologia da Informação e Engenharia de Software.

## Sobre o projeto
O Codexalab foi criado como um espaço para registrar aprendizados,
explorar conceitos técnicos e organizar conhecimentos adquiridos ao longo
dos estudos em desenvolvimento de software.

Atualmente o blog está em fase inicial e conta com poucos conteúdos,
mas será expandido gradualmente conforme novos temas forem estudados.

## Temas abordados
- Engenharia de Software
- Lógica de programação
- Desenvolvimento Web
- Fundamentos de TI

## Tecnologias
- HTML
- CSS
- JavaScript
